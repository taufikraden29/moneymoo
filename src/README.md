# Source Code Documentation

This directory contains the main source code for the MoneyMoo application.

## Folder Structure

- `assets/` - Static assets like images, fonts, etc.
- `components/` - Reusable UI components
- `hooks/` - Custom React hooks
- `lib/` - Third-party libraries and integrations (e.g., Supabase client)
- `pages/` - Route-level components
- `services/` - API and business logic services
- `utils/` - Utility functions and helpers

## Entry Point

- `main.jsx` - Application entry point with routing setup
- `App.jsx` - Main application component
- `index.css` - Global styles

## Best Practices

- Components are organized by feature in the `components/` directory
- Business logic is separated into services
- Custom hooks encapsulate reusable logic
- Utility functions are pure and side-effect free
