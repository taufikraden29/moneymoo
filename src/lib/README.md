# Libraries Documentation

This directory contains library configurations and third-party integrations for the MoneyMoo application.

## Available Libraries

### `supabaseClient.js`
Configures and exports the Supabase client with:
- Security best practices
- Session management
- Authentication state listeners
- Audit logging for security events

## Library Conventions

- Library configurations are centralized in this directory
- All third-party integrations are properly configured with security in mind
- Libraries include proper error handling and logging
- Configuration values are loaded from environment variables
